# SpringBoot 프로젝트 생성 (ver.mac)
### 사전 준비물
* Java 11 설치 (Java 11이 Springframework의 프로젝트를 생성할 때도 오류없이 잘 작동한다.)
* IDE : IntelliJ 또는 Eclipse 설치 (실무에서 Eclipse는 많이 사용하지 않는 편, 필자는 IntelliJ 사용)
------
## IntilliJ 설치 (ver.mac)
