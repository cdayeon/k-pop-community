# 인프라 구축
## 인프라 구축 목표
1. AWS EC2
  * 인스턴스 생성(8080포트 사용)
  * 공용 IP 생성
  * 서버 접속 테스트
  * 서버에서 SpringBoot 빌드 테스트
  * Swap메모리 설정
  * RDS 구축(Mysql 사용)
  * RDS 테스트
2. Jenkins로 CI/CD 구축(9090포트 사용)
  * CI/CD 파이프라인 테스트
